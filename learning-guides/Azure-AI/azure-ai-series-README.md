## How to use this series

Each guide is standalone — you can read any part independently. However,
reading in order (01 → 08) gives you the complete picture of how Azure AI
services work together in a real enterprise system.

**Beginner path:** Read 01, 02, 04 — these three cover the SmartSQL project completely.

**Expert path:** Jump to Architecture Patterns (08) and refer back to specific guides
as needed.
