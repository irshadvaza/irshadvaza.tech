# Azure Data Factory (ADF) Overview

Azure Data Factory is a cloud-based **ETL and data integration service** that allows you to create, schedule, and orchestrate data pipelines without heavy coding. It is ideal for enterprises moving data across different storage and compute systems.

## Key Features
- Pipelines: Define workflows to move and transform data.
- Datasets: Represent your data structures.
- Linked Services: Connect to different sources like Blob Storage, SQL, or Databricks.
- Triggers: Automate execution on schedules or events.
