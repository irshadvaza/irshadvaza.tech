# Azure Databricks Overview

Azure Databricks is an **analytics platform based on Apache Spark** for big data processing, machine learning, and AI workloads.

## Key Features
- Collaborative notebooks for Python, Scala, SQL
- Delta Lake for reliable, scalable data storage
- Integrated with Azure services (ADF, Power BI, ML)
- Machine Learning workflow support
